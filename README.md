Project Description:

# 1a. Installation & Run steps (github):

From your working directory run the following:
```
git clone https://github.com/olengs/Jelly-Jump.git
cd Jelly-Jump
```

# 1b. Installation via zip file:

Unzip the file and go into the working directory of the unzipped folder.

# 1c. Run steps:

Create a file called .env and add the following text to the file:

Install the required packages and run:
```
npm i
npm run app
```

Then you will now be able to visit the page via http://localhost:8000/ or http://localhost:8000/index.html.
